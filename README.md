# Requirements
**There are two ways to launch this application:**
#[1] Work with files
**Steps:**
1) run Artemis
2) run FileGenerator
3) run XmlFileReader
4) run RouteProcessor
5) run DocumentHandler
#[2] Work with classes
**Steps:**
1) run Artemis
2) run XmlClassGenerator
3) run RouteProcessor
4) run DocumentHandler
